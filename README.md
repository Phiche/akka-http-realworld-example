# RealWorld Example Application

This project has been rewritten using **Java 17** and **Spring Boot 3**.

## Getting started

1. Build and run tests
   ```bash
   mvn test
   ```
2. Start the application
   ```bash
   mvn spring-boot:run
   ```
   The server starts on `http://localhost:8080`.

The application exposes a simple `GET /articles` endpoint returning a sample article list.
